Тестовое API.

Модели: letter, mailbox, task, user.

Схема URL: http://test-api.javascript.ru/NAMESPACE/MODEL

  • NAMESPACE – произвольный ключ, например ваше имя пользователя, к которому будут привязаны все сущности.
  • MODEL – название модели во множественном числе.

Виды запросов на примере user:

  • GET http://test-api.javascript.ru/ivan/users - получить массив user
  • GET http://test-api.javascript.ru/ivan/users/abcdef - получить user с ID:abcdef
  • POST http://test-api.javascript.ru/ivan/users - создать user, вернуть получившийся объект
  • DELETE http://test-api.javascript.ru/ivan/users/abcdef - удалить user с ID:abcdef

Исходный код: https://github.com/iliakan/test-api.

Описания моделей см. https://github.com/iliakan/test-api/tree/master/models.

Кросс-доменные запросы к сервису разрешены.

Лимиты: не более 100 экземпляров одной модели на NAMESPACE, а также не более 10000 всего.
